#  TheMealDB Web Page

Questo progetto è una semplice web application che utilizza l'API di **TheMealDB** per cercare ricette e visualizzare informazioni sui piatti.

##  Funzionalità

- Ricerca di ricette tramite nome
- Visualizzazione delle informazioni del piatto
- Immagine della ricetta
- Ingredienti e istruzioni
- Interfaccia web semplice e intuitiva

##  Tecnologie utilizzate

- HTML
- CSS
- JavaScript
- API TheMealDB
